## READ BEFORE STARTING A SCENE

This prompt runs better with a little bit of setup. Before writing begins, set the engine’s operating conditions so it knows what kind of story it is producing, how hard the world should push back, and how much prose it should generate per turn.

At minimum, define:

### 1. Narrative Tone
State the intended tone of the story before play begins.
Examples:
- Soft romantic slow burn
- Gothic obsession
- Psychological thriller
- High-stakes political fantasy
- Messy dark comedy
- Intimate domestic drama
- Brutal survival horror

The engine will still preserve character fidelity and scene logic, but tone determines how the prose should feel, how pressure is framed, and how sharply consequences land.

### 2. Word Target / Word Ceiling
Set the desired response size before writing.
Examples:
- Compact: 500–800 words
- Standard: 900–1200 words
- Expansive: 1200–1500 words
- Maximum: 1500+ words when the scene demands it

If no custom ceiling is given, the preset naturally leans long and dense. Adjust this first if you do not want large multi-paragraph responses every turn.

### 3. Dark Level
Set how unforgiving the narrative should be.

- 0: Gentle. Consequences are mild, setbacks are soft.
- 1: Light pressure. Mistakes matter, but recovery is easy.
- 2: Balanced. Failure creates real complications.
- 3: Harsh. The world pushes back, trust and luck decay faster.
- 4: Severe. Recovery is slow, setbacks leave marks.
- 5: Hostile. The world is actively punishing, and failure can reshape the entire scene.

Dark Level affects consequence cost, NPC forgiveness, luck decay, and how sharply the engine escalates pressure.

### 4. Setting Law
Choose how strictly the world enforces its own rules.

- Loose:
  Genre logic bends when dramatically useful.
- Strict:
  Rules matter and breaking them carries believable consequences.
- Hostile:
  The setting punishes carelessness, shortcuts, and bad reads aggressively.

### 5. Reveal Cadence
Choose how quickly information enters the narrative.

- Sparse:
  Very little is explained. Mystery stays tight.
- Drip:
  Small pieces emerge slowly through tension and implication.
- Standard:
  Information arrives at a balanced pace.
- Aggressive:
  Revelations come fast, scenes pivot often, and hidden truths surface sooner.

Use slower cadence for suspense-heavy plots.
Use faster cadence for intrigue, escalation, and rapid plot movement.

### 6. Failure Mode
Choose what failure feels like when the user or characters misstep.

- Clean:
  The failure is clear and functional.
- Costly:
  The failure carries a tangible price.
- Humiliating:
  The failure stings personally or socially.
- Escalatory:
  The failure creates a worse problem immediately.

This controls the texture of mistakes, not whether mistakes happen.

### 7. Optional Gates
Enable only when relevant.

- Villain Gate:
  Turn on when the scene contains a true antagonist who should pressure the plot with intention and strategy.

- Intimacy Gate:
  Turn on only when the user explicitly wants explicit intimate content in the scene.

If a gate is not relevant, leave it off. The engine should not force those modules into unrelated scenes.

---

## Recommended Session Setup Format

Before the first scene, provide a setup block like this:

[NARRATIVE SETUP]

Tone:
Word Target:
Dark Level:
Setting Law:
Reveal Cadence:
Failure Mode:
Villain Gate:
Intimacy Gate:
Additional Notes:

Example:

[NARRATIVE SETUP]

Tone: Gothic obsession with polished restraint
Word Target: 1200–1500 words
Dark Level: 4
Setting Law: Strict
Reveal Cadence: Drip
Failure Mode: Costly
Villain Gate: ON
Intimacy Gate: OFF
Additional Notes: Keep the protagonist’s dignity intact even when the scene becomes threatening. Prioritize subtext, territorial behavior, and consequence-forward escalation.

---

## Good Practice

Set these controls before the story starts, and revise them whenever the project changes direction. A softer romance arc, a villain-centered thriller arc, and a punitive horror arc should not be running on the same narrative settings by accident.

The clearer the setup, the better the engine can maintain tone, pacing, consequence logic, and consistency from turn to turn.
